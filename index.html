00

要約（エグゼクティブサマリー）

最大の敵は他スクールではなく「現状維持」。ポジションはFammとSHElikesの間の空白地帯。チャネルは役割で切り分け、受け皿を整えてから拡張する。

最大の競合は「行動しないこと」

マーケティングの一次目的は比較検討の促進ではなく、行動コスト層1（放置）・層2（独学・口コミ副業）の層を層3（新しい行動＝skimama入会）へ動かす教育である。

ポジション

Famm（1ヶ月・単一スキル・シッター付き）とSHElikes（広く浅く・自走前提・学び放題）の間にある「複数職種から選べる・2ヶ月・伴走型コミュニティ・仕事獲得率93%」という空白地帯を占有する。

チャネル役割

広告＝刈り取り、LP／比較コンテンツ＝育成、Organic（IG／TikTok／YouTube）＝需要創出、LINE＝継続。現状は広告依存が強くCACが上がり続ける構造にあり、Organicへの投資シフトが急務。

意思決定の順番

新規獲得（広告拡大）より先に「受け皿」（LP・LINE後の体験・継続の仕組み）を整える。フェーズ移行は感覚ではなく定量ゲート条件で判断する。

本書で新たに具体化した点：①ポジショニング一文の型、②権威・担保軸を含む競合分解表、③ペルソナ優先度の星取り表、④Tier×ステージの9セルマトリクスと投稿配分ルール、⑤フェーズ移行の定量ゲート条件、⑥現状値→短期→中期の3点セットKPIロードマップ、⑦LINE CRMステップ配信の型、⑧リスク対応表。

01

ポジショニング一文

「誰に・何を・何で・どう差別化するか」の4点を1つのタグラインに圧縮する。

誰に
育児中で「パートに戻る／独学で迷子になる／何もせず自己肯定感を削り続ける」のいずれかに留まっている女性。特に育休復帰前後・パート収入の天井を感じている層。
何を
「収入が増えない」→「自己効力感が下がる」→「行動できずさらに収入が増えない」という悪循環を、"最初の案件獲得"という具体的な成功体験で断ち切る。
何で
ママ限定コミュニティ × メンター伴走 × 複数職種から選べる柔軟さ × 2ヶ月という現実的な期間設計の掛け合わせ。
どう差別化
Fammのように単一スキルに賭けさせず、SHElikesのように自走を前提にしない。「同じ境遇の仲間と伴走してもらいながら、自分に合った職種で、仕事を獲得するところまで並走してもらえる」という、競合が同時には満たせない二軸（伴走の深さ×職種選択の自由）を両取りする。

「意識が高い系」でも「怪しい副業」でもない、同じ“普通のママ”が伴走してくれる在宅ワークの実家。

03

ペルソナ優先度・星取り表

Tier仮説を、入口の悩みと刺さる訴求順序まで具体化したもの。

優先度ペルソナ痛みの強度入口 → 中間 → 出口
1st 育休復帰前後で「パートに戻るしかないのか」と悩む層／パート収入の天井に限界を感じている層 ★★★★★ 「このままパートに戻ったら何を失うか」の可視化 → 獲得率93%・職種を選べる自由 → 分割12,000円〜・無料勉強会でのゼロリスク体験
2nd 専業主婦で「そろそろ何か始めたい」と漠然と思っている層 ★★★★☆ 「自己肯定感を取り戻す」共感コンテンツ → 同じ境遇のママ仲間・メンター伴走の安心 → 無料勉強会（特典：在宅ワークママBOOK）
3rd 小学生以上の子を持ち「小1の壁」に直面／地方在住で働き口が限られる層 ★★★★☆ 「小1の壁」あるあるの共感 → フルオンライン・時間拘束のなさ → 職種診断・卒業生実例
4th 育休中で復職に迷いがある層（将来層としてのフォロー） ★★★☆☆ 産後キャリア不安の社会課題化コンテンツ → 将来の選択肢としての在宅ワーク認知 → LINE登録によるナーチャリング開始

同じ職種訴求でも、1st層には収入天井・時間拘束という機能的ペインで即効性を、2nd・4th層には自己肯定感の回復という情緒的ペインで安全性・仲間感を先に見せる。

06

フェーズ移行の定量ゲート条件

「Organic強化 → 広告投資配分の見直し」という後戻りしにくい意思決定を、感覚でなく基準で判断するための仮ゲート仮説。社内データ取得後に閾値を確定させる。

Phase 1 → Phase 2（土台構築 → スケール）

  1. アカウントコンセプト・プロフィール設計が確定済みか（Yes/No）
  2. 初期50〜100本の投稿分析が完了しているか（Yes/No）
  3. リール平均視聴維持率が一定水準（要ベースライン取得）を超えているか（数値基準）
  4. LINE登録への導線が実装済みか（Yes/No）

4項目中3項目以上でPhase 2へ。未達なら土台構築を延長。

Phase 2 → Phase 3（スケール → 転換最大化）

  1. Organic経由のLINE登録数が広告経由CPAを下回るCPAで安定しているか（数値基準）
  2. 卒業生発信（UGC）が月次で一定件数発生しているか（数値基準）
  3. LINE登録者数が母数として転換施策に耐える規模に達しているか（数値基準）

3項目中2項目以上でPhase 3へ。

07

LINE CRMステップ配信の型

  • Day 0
    ウェルカムメッセージ＋在宅ワークママBOOKの受け渡し（緊張・不安の解消）
  • Day 3〜5
    無料勉強会の日程案内・参加ハードルを下げるFAQ（子供がいても大丈夫、等）
  • Day 7前後
    勉強会参加者へ卒業生実例・獲得率93%の根拠提示（比較検討の後押し）
  • Day 14前後
    未参加・未申込者への再アプローチ（限定特典 or 個別相談案内）

リッチメニュー4象限案：①無料勉強会に申し込む／②卒業生の声を見る／③よくある質問／④公式Instagramを見る（Organicとの回遊導線）。

KPIロードマップ：現状値 → 短期 → 中期

フェーズ期間目的KPI（現状 → 短期 → 中期）
Phase 1
土台構築		 〜3ヶ月 アカウントのジャンル認識を確立し、初速データを取る フォロワー増数・視聴維持率：現状値未取得（最初のタスク） → 社内合意の初期目標 → 業界水準比での中期目標

スケール		 3〜9ヶ月 Organic経由のLINE登録を主要KPIに切り替える LINE登録数・登録CPA：現状の広告CPAを基準値とし、Organic経由CPAがそれを下回ることを短期目標に設定
Phase 3
転換最大化		 9〜18ヶ月 LINE登録→勉強会予約→成約の転換率を上げる 勉強会予約率・成約率：現状値（社内データ要取得）→ 短期は現状比+α → 中期はCAC全体の低下で評価
Phase 4
資産化		 18ヶ月〜 指名検索・紹介が拡大し広告依存から脱却する 自然流入比率・紹介経由成約率：現状値取得 → 短期目標 → 「売上10億円」逆算からの中期目標
最初のタスク現状値の取得そのもの。現状値なしに立てた目標値は仮の仮説にすぎない。
09

リスクと対応方針

リスク影響対応方針
現状値（フォロワー・LINE登録・CPA等）が未取得のままKPI目標を確定してしまう目標が非現実的になり、投資判断を誤る着手初週で社内データ・既存アカウント現状の棚卸しを最優先タスク化
「獲得率93%」等の実績数値の算出根拠が未確認外部発信時に景表法リスク、社内外の信頼低下発信前に算出定義・母数を確認し、根拠を開示できる形に整理
Famm・SHElikesの実クリエイティブ未確認のまま訴求を推定で作る差別化訴求が的外れになるリスクMeta広告ライブラリでの実物収集を早期に実施
メンター品質がスケール時に毀損する獲得率93%・満足度99.3%というブランド最大の資産が崩れるメンター1名あたり受講生数・品質指標をKPIロードマップに組み込み、拡大速度をゲート条件で制御
提携先「フジ子さん」がスクール経由をバイパスし直接競合化する卒業後の案件獲得導線が弱まる提携関係の位置づけ（送客 vs 競合）を早期に整理
Organic投稿がフォロワー数の最大化だけを目的化するLINE登録・勉強会予約という本来のKPIから逸れるフォロワー数を非目的化し、LINE登録転換数を常に主指標として運用会議で確認

次のアクション

要検証事項の中でも着手優先度が高いもの。

  • Organicアカウント現状値（フォロワー数・エンゲージメント・LINE登録経由内訳）のベースライン取得
  • 実績数値（96.0%／99.3%／93.0%）の算出定義・母数の社内確認
  • Famm・SHElikesの実広告クリエイティブ（Meta広告ライブラリ）の収集
  • 卒業生インタビューによるペイン層4インサイト・JTBDの一次検証
  • 「フジ子さん」提携の位置づけ整理（送客 vs 将来的競合）
