§0

要約（エグゼクティブサマリー）

最大の敵は他スクールではなく「現状維持」。マーケティングの一次目的は比較検討促進ではなく、行動コスト層1（放置）・層2（独学・口コミ副業）にいる層を層3（新しい行動＝skimama入会）へ動かす教育である。

  • ポジション：Famm（1ヶ月・単一スキル・シッター付き）とSHElikes（広く浅く・自走前提・学び放題）の間にある「複数職種から選べる・2ヶ月・伴走型コミュニティ・仕事獲得率93%」という空白地帯を占有する。
  • チャネル役割：広告＝刈り取り、LP/比較コンテンツ＝育成、Organic（IG/TikTok/YouTube）＝需要創出、LINE＝継続。現状は広告依存が強くCACが上がり続ける構造にあり、Organicへの投資シフトが急務（brain/09 §4・§5）。
  • 意思決定の順番：新規獲得（広告拡大）より先に「受け皿」（LP・LINE後の体験・継続の仕組み）を整える。フェーズ移行は感覚でなく定量ゲート条件で判断する（§6）。
本書で新たに具体化した点

①ポジショニング一文の型、②権威・担保軸を含む競合分解表、③ペルソナ優先度の星取り表、④Tier×ステージの9セルマトリクスと投稿配分ルール、⑤フェーズ移行の定量ゲート条件、⑥3点セット（現状値→短期→中期）のKPIロードマップ、⑦LINE CRMステップ配信の型、⑧リスク対応表。

§1

ポジショニング一文（brain/09 §1 の型で圧縮）

誰に
育児中で「パートに戻る／独学で迷子になる／何もせず自己肯定感を削り続ける」のいずれかに留まっている女性。特に育休復帰前後・パート収入の天井を感じている層
何を
「収入が増えない」→「自己効力感が下がる」→「行動できずさらに収入が増えない」という悪循環を、“最初の案件獲得”という具体的な成功体験で断ち切る
何で
ママ限定コミュニティ × メンター伴走 × 複数職種から選べる柔軟さ × 2ヶ月という現実的な期間設計の掛け合わせ
どう差別化
Fammのように単一スキルに賭けさせず、SHElikesのように自走を前提にしない。「同じ境遇の仲間と伴走してもらいながら、自分に合った職種で、仕事を獲得するところまで並走してもらえる」という、競合が同時には満たせない二軸（伴走の深さ×職種選択の自由）を両取りする

「意識が高い系」でも「怪しい副業」でもない、同じ“普通のママ”が伴走してくれる在宅ワークの実家。— 一言タグライン仮説（01_戦略/02_マーケティング戦略.md §4より再掲）

§3

ペルソナ優先度・星取り表（brain/09 §3 の型）

CLAUDE.mdのTier仮説を、入口の悩みと刺さる訴求順序まで具体化。

優先度ペルソナ痛みの強度入口→中間→出口の訴求順序
1st 育休復帰前後で「パートに戻るしかないのか」と悩む層／パート収入の天井に限界を感じている層 ★★★★★ 入口：「このままパートに戻ったら何を失うか」の可視化 → 中間：獲得率93%・職種を選べる自由 → 出口：分割12,000円〜・無料勉強会でのゼロリスク体験
2nd 専業主婦で「そろそろ何か始めたい」と漠然と思っている層 ★★★★☆ 入口：「自己肯定感を取り戻す」共感コンテンツ → 中間：同じ境遇のママ仲間・メンター伴走の安心 → 出口：無料勉強会（特典：在宅ワークママBOOK）
3rd 小学生以上の子を持ち「小1の壁」に直面／地方在住で働き口が限られる層 ★★★★☆ 入口：「小1の壁」あるあるの共感 → 中間：フルオンライン・時間拘束のなさ → 出口：職種診断・卒業生実例

ポイント（brain/09 §3準拠）

1st層には「収入天井・時間拘束」という機能的ペインで即効性を、2nd・4th層には自己肯定感の回復という情緒的ペインで安全性・仲間感を先に見せる。同じ職種訴求でも入口の一言を変える。

§6

フェーズ移行の定量ゲート条件（brain/09 §6）

「Organic強化 → 広告投資配分の見直し」という後戻りしにくい意思決定を、感覚でなく基準で判断するための仮ゲート（社内データ取得後に閾値を確定させる）。

Phase 1 → Phase 2 移行条件

土台構築 → スケール

  1. アカウントコンセプト・プロフィール設計が確定済みか（Yes/No）
  2. 初期50〜100本の投稿分析（brain/06準拠）が完了しているか（Yes/No）
  3. リール平均視聴維持率が一定水準（要ベースライン取得）を超えているか（数値基準）
  4. LINE登録への導線（プロフィールリンク等）が実装済みか（Yes/No）
通過ライン：3 / 4 項目

→ 4項目中3項目以上でPhase 2へ。未達なら土台構築を延長。

Phase 2 → Phase 3 移行条件

スケール → 転換最大化

  1. Organic経由のLINE登録数が広告経由CPAを下回るCPAで安定しているか（数値基準）
  2. 卒業生発信（UGC）が月次で一定件数発生しているか（数値基準）
  3. LINE登録者数が母数として転換施策に耐える規模に達しているか（数値基準）
通過ライン：2 / 3 項目

→ 3項目中2項目以上でPhase 3へ。

使いどころ

「Web広告依存から脱却する」という意思決定を社内で通す武器として、この基準を実データで埋めてから経営会議に提示する（brain/09 §6の趣旨）。

§8

KPIロードマップ：現状値→短期→中期の3点セット（brain/09 §12）

01_戦略/02_マーケティング戦略.md §8のフェーズ表に、現状値の欠落自体を最初のタスクとして明示する形で数値枠を追加。

フェーズ期間目的（1行で言い切る）主要施策KPI：現状値 → 短期目標 → 中期目標
Phase 1：土台構築 〜3ヶ月 アカウントのジャンル認識を確立し、初速データを取る コンセプト確定、初期投稿分析50〜100本 フォロワー増数・視聴維持率：現状値未取得（最初のタスク） → 社内合意の初期目標 → 業界水準比での中期目標
Phase 2：スケール 3〜9ヶ月 Organic経由のLINE登録を主要KPIに切り替える 拡散パターン確立、UGC仕組み化 LINE登録数・登録CPA：現状の広告CPAを基準値とし、Organic経由CPAがそれを下回ることを短期目標に設定
Phase 3：転換最大化 9〜18ヶ月 LINE登録→勉強会予約→成約の転換率を上げる LINEシナリオ最適化 勉強会予約率・成約率：現状値（社内データ要取得）→ 短期は現状比+α → 中期はCAC全体の低下で評価
Phase 4：資産化 18ヶ月〜 指名検索・紹介が拡大し広告依存から脱却する コミュニティ・卒業生ネットワーク可視化 自然流入比率・紹介経由成約率：現状値取得 → 短期目標 → 「売上10億円」逆算からの中期目標
最初のタスク

最初のタスクは「現状値の取得」そのもの。現状値なしに立てた目標値は仮の仮説にすぎない（§9参照）。

§9

リスクと対応方針表（brain/09 §13）

リスク影響対応方針
現状値（フォロワー・LINE登録・CPA等）が未取得のままKPI目標を確定してしまう 目標が非現実的になり、Organic投資の意思決定を誤る 着手初週で社内データ・既存アカウント現状の棚卸しを最優先タスク化
「獲得率93%」等の実績数値の算出根拠が未確認 外部発信時に景表法リスク、社内外の信頼低下 発信前に算出定義・母数を確認し、根拠を開示できる形に整理（brain/09 §9エビデンス戦略）
Famm・SHElikesの実クリエイティブ未確認のまま訴求を推定で作る 差別化訴求が的外れになるリスク Meta広告ライブラリでの実物収集を早期に実施（01_競合リサーチ.md §8）
メンター品質がスケール時に毀損する 獲得率93%・満足度99.3%というブランド最大の資産が崩れる メンター1名あたり受講生数・品質指標をKPIロードマップに組み込み、拡大速度をゲート条件で制御（§6）
提携先「フジ子さん」がスクール経由をバイパスし直接競合化する 卒業後の案件獲得導線が弱まる 提携関係の位置づけ（送客 vs 競合）を早期に整理（CLAUDE.md要検証事項）
Organic投稿がフォロワー数の最大化だけを目的化する LINE登録・勉強会予約という本来のKPIから逸れる フォロワー数を非目的化し、LINE登録転換数を常に主指標として運用会議で確認
